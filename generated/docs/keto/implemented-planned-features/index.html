Version: v0.6

Implemented and Planned Features

In October 2020 we decided to rewrite Ory Keto from scratch, implementing state-of-the-art access control and permissions. We decided to make Ory Keto the first Open Source implementation of Google's Zanzibar paper.

This document gives an overview of all implemented and planned features. Unfortunately we cannot give delivery time estimates for Open Source products, but you can help us to accelerate development by supporting Ory.

Implemented Features#

  • Read-API
  • Write-API
  • Check-API
  • Expand-API
  • Namespaces

Next Upcoming Features#

  • Subject Set rewrites
  • Native ABAC & RBAC Support
  • Integration with other policy servers

Planned Features#

  • Database sharding
  • Namespace-specific database settings
  • Cockroach optimized persistence layer
  • Latency reduction through aggressive caching

... and many more, tracked as issues
